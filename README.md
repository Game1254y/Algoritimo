# Algoritimos em Python

Este repositório contém meus primeiros exercícios de programação em Python desenvolvidos durante o início da faculdade de Análise e Desenvolvimento de Sistemas.

## Exercícios

* Cálculo de desconto
* Cálculo de IMC
* Verificação de maior número
* Verificação de ano bissexto
* Sistema simples de login
* Média final de aluno
* Verificação de idade para dirigir
* Números pares
* Verificação de idade

## Objetivo

Praticar conceitos básicos de programação como:

* `input`
* `if`, `elif`, `else`
* operações matemáticas
* lógica de programação
